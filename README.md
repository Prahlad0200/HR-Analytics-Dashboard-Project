# HR-Analytics-Dashboard-Project
I developed an HR Analytics Dashboard using Power BI to analyze employee attrition and workforce trends. The main objective was to identify key factors affecting employee turnover.

I used employee dataset containing attributes like age, salary, education, job role, and years at company.

In the dashboard, I created KPIs such as total employees, attrition count, attrition rate, average salary, and average tenure.

I performed analysis across multiple dimensions like age group, salary slab, education background, and job roles.

From the insights, I found that attrition is highest among employees with low salary, early tenure, and in roles like sales executive and lab technician.

I also added interactive filters for department-level analysis to make the dashboard dynamic.

This project helps HR teams take data-driven decisions to improve employee retention.
